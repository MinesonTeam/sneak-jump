## 🧪 Code Quality & CI/CD
SneakJump strictly adheres to Enterprise coding standards. Every Pull Request is automatically scanned by **SonarQube** and **Codacy** to ensure:
* `0` Security Vulnerabilities
* `0` Memory Leaks
* `Grade A` Maintainability & Reliability

We use static application security testing (SAST) to prevent bugs before they reach production.
